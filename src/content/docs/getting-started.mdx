To use Corsfix, add `https://proxy.corsfix.com/?` before your URL when making fetch requests.

## Basic Usage

```javascript
// Instead of direct fetch
fetch("https://api.example.com/data");

/*
  Add Corsfix URL before the API
  https://proxy.corsfix.com/? + https://api.example.com/data
*/
fetch("https://proxy.corsfix.com/?https://api.example.com/data");
```

<Aside>
Make sure to add your domain in the [dashboard](/docs/dashboard/application) to use Corsfix on live websites.

For local websites (localhost), you can use Corsfix without registration.
</Aside>

## Example

```javascript
// GET request
fetch("https://proxy.corsfix.com/?https://api.example.com/users")
  .then((response) => response.json())
  .then((data) => console.log(data));

// POST request
fetch("https://proxy.corsfix.com/?https://api.example.com/users", {
  method: "POST",
  body: JSON.stringify({ name: "John" }),
  headers: {
    "Content-Type": "application/json",
  },
});
```
