<Aside>
Make sure to add your domain in the [dashboard](https://app.corsfix.com/applications) to use Corsfix on live websites. <br/>
For local websites (localhost), you can use Corsfix without any setup.
</Aside>

To use Corsfix, add `https://proxy.corsfix.com/?` before your URL when making fetch requests.

## Basic Usage

```javascript
// Instead of direct fetch
fetch("https://api.example.com/data");

/*
  Add Corsfix URL before the API
  https://proxy.corsfix.com/? + https://api.example.com/data
*/
fetch("https://proxy.corsfix.com/?https://api.example.com/data");
```

## Example

```javascript
// GET request
fetch("https://proxy.corsfix.com/?https://api.example.com/users")
  .then((response) => response.json())
  .then((data) => console.log(data));

// POST request
fetch("https://proxy.corsfix.com/?https://api.example.com/users", {
  method: "POST",
  body: JSON.stringify({ name: "John" }),
  headers: {
    "Content-Type": "application/json",
  },
});
```
